Privacy Policy for Amazon Sensitive Word Detection Tool

Effective Date: March 25, 2025

1. Introduction

This privacy policy applies to the Amazon Sensitive Word Detection Tool Chrome extension (hereafter "the Extension"). We are committed to protecting your data privacy through local processing and zero data collection principles[6,7](@ref).

2. Data Storage & Usage

2.1 Local Storage

  • Sensitive word libraries and configurations are stored exclusively in browser local storage (chrome.storage.local)
  • All data remains on your device and is never uploaded to servers[9](@ref)

2.2 Document Processing

  • Uploaded documents (.txt/.doc/.pdf etc.) are temporarily parsed in memory
  • File contents are destroyed immediately after analysis[7](@ref)

3. Permission Declaration

Permission Purpose Data Scope
storage Save custom word libraries & settings Local device storage
activeTab Analyze current webpage content Current tab only
scripting Dynamic text highlighting Non-modification of original content
fileSystem.read Process user-selected documents Temporary memory access

4. User Rights

  • Complete control over local data through extension settings
  • One-click data purge functionality
  • No third-party tracking or analytics[9](@ref)

5. Policy Updates

Changes will be notified via extension update logs. The latest version is always available on our Chrome Web Store page.
